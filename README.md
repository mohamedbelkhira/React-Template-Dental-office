## Getting Started

1. Install packages

```
yarn install
```

2. Run project

```
yarn start
```


3. Run Desktop application

```
npm run electron-dev
```


## Technology stack

- [Material UI V5](https://mui.com/core/)
- Built with React Hooks API.
- React context API for state management.
- SWR.
- React Router for navigation routing.
- Support for Vite.
- Code splitting.
- CSS-in-JS.
